# Auto-Roco 洛克王国自动化助手

这是一个基于图像识别（OpenCV）的游戏自动化辅助工具，专门为《洛克王国：世界》设计。它能够自动识别战斗状态，并根据用户选择执行“聚能模式”或“逃跑模式”。

---

## 🚀 核心功能
*   **聚能模式 (Mode 1)**：在进入战斗时，自动模拟键盘点击 `X` 键。
*   **逃跑模式 (Mode 2)**：在进入战斗时，自动按 `ESC` 键并模拟鼠标点击“是”按钮。
*   **后台监控**：支持游戏窗口在被遮挡时依然进行图像识别（只要窗口不最小化）。

---

## 📖 使用步骤

### 第一步：环境配置
确认你的电脑已安装 **Python 3.10** 或更高版本。可以通过在终端输入 `python --version` 查看。

### 第二步：以管理员身份打开终端
1.  右键点击 Windows 开始菜单或搜索框。
2.  选择 **Windows PowerShell (管理员)** 或 **终端 (管理员)**。
3.  **注意**：为了确保脚本能向游戏发送点击指令，必须使用管理员模式。

### 第三步：安装依赖库
定位到本项目文件夹，运行以下命令安装必要的运行库：
```bash
pip install opencv-python numpy mss keyboard pywin32
```

### 第四步：启动脚本
在管理员终端中运行：
```bash
python main.py
```
随后根据屏幕提示输入 `1` 或 `2` 选择功能模式。

---

## 💡 注意事项
*   **分辨率要求**：脚本默认适配 `2560x1600` 分辨率。
*   **停止方式**：在任何时候按下键盘上的 **`F8`** 键即可立即安全退出。
*   **窗口状态**：游戏窗口可以被其他窗口覆盖，但**不能最小化**到任务栏，否则图像识别将失效。

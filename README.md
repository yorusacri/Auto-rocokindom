# Auto-Roco 洛克王国自动化助手

这是一个基于图像识别（OpenCV）的游戏自动化辅助工具，专门为《洛克王国：世界》设计。它能够自动识别战斗状态，并根据用户选择执行“聚能模式”“逃跑模式”“污染计数模式”或“智能模式”。

## 🚀 核心功能
*   **聚能模式 (Mode 1)**：在进入战斗时，自动模拟键盘点击 `X` 键。
*   **逃跑模式 (Mode 2)**：在进入战斗时，自动按 `ESC` 键并点击“是”按钮。
    *   统一使用**物理鼠标点击**。
    *   执行逃跑点击时，请确保“是”按钮露出，不能被其他窗口遮挡。
*   **污染计数模式 (Mode 3)**：仅进行检测与计数，不执行任何自动操作。
    *   终端展示污染次数（内部仍保留战斗计数用于战斗区间识别）。
*   **智能模式 (Mode 4)**：每场战斗开始时进行一次类型判定。
    *   污染战斗：执行聚能（按 `X`）。
    *   普通战斗：执行逃跑（`ESC` + 点击“是”按钮）。
*   **后台监控**：支持游戏窗口在被遮挡时依然进行图像识别（只要窗口不最小化）。

## ⭐ 建议用法

1. 好友**刷异色**时，上一个属性抵抗目标精灵攻击的精灵（防止暴毙），使用聚能模式，保持窗口非最小化挂机即可。期间其他窗口可以任意覆盖游戏窗口，不妨碍使用设备干其他事。

2. 好友**跑图刷花刷矿**时，可以开逃跑模式，也可以开聚能模式，但需要和好友协商好。

## 📖 使用步骤

### 第一步：环境配置
确认你的电脑已安装 **Python 3.10** 或更高版本。可以通过在终端输入 `python --version` 查看。

### 第二步：以管理员身份打开终端
1.  右键点击 Windows 开始菜单或搜索框。
2.  选择 **Windows PowerShell (管理员)** 或 **终端 (管理员)**。
3.  **注意**：为了确保脚本能向游戏发送点击指令，必须使用管理员模式。

### 第三步：安装依赖库
定位到本项目文件夹，运行以下命令安装必要的运行库：
```bash
pip install opencv-python numpy mss keyboard pywin32
```

### 第四步：启动脚本
在管理员终端中运行：
```bash
python main.py
```
随后根据屏幕提示输入 `1`、`2`、`3` 或 `4` 选择功能模式。

## 💡 注意事项
*   **分辨率建议**：本脚本已实现**自适应分辨率**功能，能够根据游戏窗口大小自动缩放识别模板。
    *   **推荐分辨率**：**2K（2560x1600 或 2560x1440）**。分辨率越高，画面细节越丰富，识别精度越高。
    *   **精度提示**：在低分辨率下，由于像素损失，识别分数 (Score) 会降低。我们已将触发阈值调低至 **0.5** 以兼容低端设备。
    *   **进阶适配**：如果你的分辨率非常特殊导致脚本失效，你可以参考 `templates/` 目录下的截图格式，**在你自己的分辨率下重新截图**并覆盖 `templates/` 中的同名文件（如 `skill_main.png`），脚本将完美适配你的专属环境。
*   **停止方式**：在任何时候按下键盘上的 **`F8`** 键即可立即安全退出。
*   **窗口状态**：游戏窗口可以被其他窗口覆盖，但**不能最小化**到任务栏，否则图像识别将失效。
*   **逃跑模式点击要求**：逃跑模式使用物理点击，请确保点击瞬间“是”按钮可见且未被遮挡；若被覆盖，点击会落在前景窗口上。
*   **审计日志**：关键节点会写入 `logs/audit.log`（JSON 格式），便于快速回顾。
    *   常见事件：`模式已选择`、`聊天模板缺失`、`战斗次数增加`、`智能模式判型`、`逃跑确认点击成功`、`逃跑确认点击失败`。
    *   设计目标：减少冗余字段，优先保留“发生了什么”的关键结果信息。
